Build Syncloud Device Yourself

Building Syncloud device is easy. You will need some hardware: single-board computer, power adapter, ethernet cabe, SD card. You will also need a computer with SD card reader.

Just follow the instruction below.

Install Software


Create microSD card (4GB or more) with Syncloud image:

1Uncompress downloaded image file:

Linux: unxz [image file]

Windows: 7-zip

Mac OS: The Unarchiver

2Write image to SD card

Linux:
Identify SD card and write image to it:
lsblk
sudo dd if=./[image file] of=[SD card device]

Mac OS:
Identify SD card and write image to it:
diskutil list
dd if=./[image file] of=[SD card device]

Windows:
Use Win32 Disk Imager

3Insert SD card into device

Your Syncloud device is ready. Follow main instruction to activate it.

Here are SSH credentials for the device. Login: root, password: syncloud
