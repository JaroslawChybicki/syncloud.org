Install Software


Create microSD card (4GB or more):

1Uncompress image file (Linux):

unxz [image file]

Or use 7-zip on Windows

Or use The Unarchiver on Mac

2Write image to SD card (Linux/Mac):

Identify sd card (Mac only): diskutil list

Unmount (Mac only): diskutil unmountDisk /dev/mmcblk0

sudo dd if=./[image file] of=/dev/mmcblk0

Or use Win32 Disk Imager on Windows

3Insert SD card into device

4Power on

SSH login / password: root / syncloud
