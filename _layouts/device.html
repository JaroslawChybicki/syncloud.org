Build your own online storage with
{{ page.boardname }}

1. Get Hardware

Buy {{page.boardname}} single-board computer and {{page.storage-type}} hard drive.

2. Get Software

Download syncloud image for {{page.boardname}}.

3. Install Software


Create microSD card (2GB or more):

1Uncompress image file (Linux/Mac):

unxz {{page.imagefile}}.xz

Or use 7-zip on Windows

2Write image to SD card (Linux/Mac):

diskutil unmountDisk /dev/mmcblk0(required on Mac only)

sudo dd if=./{{page.imagefile}} of=/dev/mmcblk0

Or use Win32 Disk Imager on Windows

3Insert SD card into device

4Power on

Syncloud image is based on {{page.base-image-name}}

SSH login / password: root / syncloud

4. Connect Everything


Connect hard disk drive to the device. You can use SATA or USB connector. Don't forget to create ext4 partition on the drive.
Connect your device to your home router. Use one of the available LAN ports.
Turn the device on.

5. Activate


Use Syncloud Android application to activate your device, link it with domain name at syncloud.it and enable ownCloud. Watch video for this step.

6. Use It!


Now you ready to use your online storage! Access. Sync. Share. Learn more about ownCloud features.


Download ownCloud client application for your platform:

Windows Mac OS X Linux Android iPhone
