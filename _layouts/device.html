Build your own online storage with
{{ page.boardname }}

1. Compatibility Check


Use Syncloud Android application to check if your router is compatible. Check is automatically performed on first run and always available on Settings screen.

2. Get Hardware

Buy {{page.boardname}} single-board computer
and {{page.storage-type}} hard drive.

3. Get Software

Download syncloud image
for {{page.boardname}}.

4. Install Software


Create microSD card (4GB or more):

1Uncompress image file (Linux):

unxz {{page.imagefile}}.xz

Or use 7-zip on Windows

Or use The Unarchiver on Mac

2Write image to SD card (Linux/Mac):

Identify sd card (Mac only): diskutil list

Unmount (Mac only): diskutil unmountDisk /dev/mmcblk0

sudo dd if=./{{page.imagefile}} of=/dev/mmcblk0

Or use Win32 Disk Imager on Windows

3Insert SD card into device

4Power on

Syncloud image is based on {{page.base-image-name}}

SSH login / password: root / syncloud

5. Connect Everything


1Connect hard disk drive to the device with USB cable.

2Try to use dedicated HDD power cable.

3Don't forget to create single partition on the drive (ext4 is tested).

4Connect your device to your home router. Use one of the available LAN ports.

5Turn the device on.

5. Activate


Use Syncloud Android application to activate your device, link it with domain name at syncloud.it and enable ownCloud. Watch video for this step.

7. Use It!


Now you ready to use your online storage! Access. Sync. Share. Learn more about ownCloud features.


Download ownCloud client application for your platform:

Windows Mac OS X Linux Android iPhone
